# 작업 5 — 네비게이션: 상단 메뉴 유지 + 모바일 하단 메뉴 신설 (pudufu 식)

**상단 메뉴는 그대로 두고**(강의/전자책/전문가), 모바일 전용 **하단 메뉴를 별도로** `홈 / 검색 / 컨텐츠 / 내강의`로
구성한다. 참고: <https://pudufu.co.kr/home/pdf_premium> (모바일 하단 4메뉴).

> ⚠️ 앞 버전 오해 정정: 상단 메뉴를 홈/검색/컨텐츠/내강의로 **바꾸는 게 아니다.** 상단(데스크톱)은
> 강의·전자책·전문가를 **유지**하고, **하단(모바일)은 새 메뉴**다. **전문가는 상단 메뉴에 그대로 둔다**
> (드롭다운 이동 취소).

## 현재 상태 — [src/components/AcademyLayout.tsx](../../src/components/AcademyLayout.tsx)
- 상단 메뉴 `MENU`(`:6-11`): `강의 / 전자책 / 전문가 / 관심 강의`. `hidden … md:block`(`:63`) = **데스크톱 전용**.
- 모바일 전용 검색바 줄(`:48-59`).
- 모바일 하단 탭바(`:88-113`): `홈/강의/전자책/전문가/관심`(`grid-cols-5`).
- 데스크톱 중앙 헤더 검색창(`:32-43`).

## 변경

### 1) 상단 메뉴(데스크톱) — 유지 + 라벨 한 개만 교체
`MENU`(`:6`)에서 **강의·전자책·전문가는 그대로**, 마지막 `관심 강의`만 → **`내 강의`(`/my`)** 로 교체.
```ts
const MENU = [
  { to: '/library', label: '강의' },
  { to: '/ebooks',  label: '전자책' },
  { to: '/experts', label: '전문가' },
  { to: '/my',      label: '내 강의' },
]
```
- `hidden … md:block`(`:63`) **유지**(상단 메뉴는 데스크톱 전용 그대로).

### 2) 모바일 하단 메뉴 — 새 4항목으로 교체
하단 탭바(`:88-113`)의 항목을 교체(개수 5→4, `grid-cols-5`→`grid-cols-4`):
| 라벨 | 경로 | 아이콘(예) |
|---|---|---|
| 홈 | `/` | 🏠 |
| 검색 | `/search` | 🔍 |
| 컨텐츠 | `/content` | 📚 |
| 내강의 | `/my` | 🎒 |
- 기존 "강의/전자책 상세에서 하단탭 숨김"(`isReaderDetail`) 로직은 유지. 결제 페이지 하단탭 숨김은
  [02-mobile-footer-layout.md](./02-mobile-footer-layout.md)에서 처리.

### 3) 모바일 상단 검색바 제거
- 모바일 전용 검색 줄(`:48-59`) **삭제**. 모바일 검색은 하단 "검색" 항목으로만.

### 4) 데스크톱 헤더 중앙 검색창 — 통합검색으로 연결
- `:32-43` 검색창은 **유지**하되 동작을 `/search`로 연결([07-global-search.md](./07-global-search.md)).
  (이게 데스크톱의 유일한 검색 진입점.)

### 5) 전문가 — 변경 없음
- 상단 메뉴에 **그대로 유지**. (앞 버전의 "프로필 드롭다운 이동"은 **취소**.)

## 데스크톱 vs 모바일 동선 요약
| | 데스크톱(상단) | 모바일(하단) |
|---|---|---|
| 강의 | 강의(/library) | (컨텐츠에 통합) |
| 전자책 | 전자책(/ebooks) | (컨텐츠에 통합) |
| 전문가 | 전문가(/experts) | (홈/검색에서 도달) |
| 통합 | — | 컨텐츠(/content) |
| 검색 | 헤더 중앙 검색창 → /search | 검색 → /search |
| 내 강의 | 내 강의(/my) | 내강의(/my) |

> 컨텐츠(/content)는 **모바일 진입 전용 통합 페이지**([08-content-hub.md](./08-content-hub.md)). 데스크톱은 강의/전자책을
> 상단 메뉴로 따로 본다.

## 검증
- 데스크톱 상단 메뉴 = `강의 · 전자책 · 전문가 · 내 강의`(전문가 유지).
- 모바일 하단 메뉴 = `홈 · 검색 · 컨텐츠 · 내강의`(4칸), 상단 검색바 없음.
- 모바일 "컨텐츠" → `/content`(강의+전자책 통합), "검색" → `/search`, "내강의" → `/my`.
- 데스크톱 헤더 검색 엔터 → `/search`.
- 강의/전자책 상세·결제에서 하단 메뉴 숨김 규칙 정상.
- `npm run build` 통과.
