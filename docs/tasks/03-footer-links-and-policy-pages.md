# 작업 3 — 푸터 단순화 + 정책/문의 페이지 신설 (그래플레이 식)

푸터에서 **페이지가 없는 메뉴를 삭제**한다. 구체적으로 **"회사" 칼럼(소개/공지/FAQ) 전체**와
**"전문가 되기"** 를 제거. **강의 둘러보기·전문가**(이미 페이지 있음)와 **고객지원(문의하기/이용약관/
개인정보처리방침)** 은 남긴다. 정책/문의 본문은 그래플레이 본 프로젝트(`../grapplay/pages/`)에서 가져와
비즈에 맞게 조정(구독 제외). **환불정책은 별도 페이지가 아니라 약관 안**(제6조의2)에 둔다.

## 최종 푸터 구성 (2칼럼)
```
서비스      : 강의 둘러보기 → /library,  전문가 → /experts        (둘 다 기존 페이지)
고객지원    : 문의하기 → /contact,  이용약관 → /terms,  개인정보처리방침 → /privacy   (3개 신설)
```
+ 사업자 정보 블록(상호명/대표/사업자번호/주소/이메일/전화) 유지.
> 참고: 그래플레이 푸터는 문의/약관/개인정보 3개만(`../grapplay/components/Layout.tsx:736-738`). 비즈는
> 여기에 강의 둘러보기·전문가만 더 둔다.

## 현재 상태 — [src/components/AcademyLayout.tsx](../../src/components/AcademyLayout.tsx)
- 푸터 항목이 **링크가 아니라 텍스트**(`FooterCol`이 `string[]`을 `<span>`으로, `:261`).
  - `:128` 서비스: `강의 둘러보기 / 전문가 / 전문가 되기`
  - `:129` 회사: `소개 / 공지사항 / FAQ`
  - `:130` 고객지원: `문의하기 / 이용약관 / 개인정보처리방침`
- 사업자 정보 블록(`:135-151`)은 이미 있음(유지).

## 변경

### 1) 푸터 단순화 ([AcademyLayout.tsx:127-131](../../src/components/AcademyLayout.tsx#L127))
- **삭제**: "회사" 칼럼(`:129` 소개/공지/FAQ) **통째로**, "서비스" 칼럼에서 **"전문가 되기"** 1개.
- **유지**: 서비스(강의 둘러보기/전문가) 2개, 고객지원(문의하기/이용약관/개인정보처리방침) 3개 → **2칼럼**.
- `FooterCol`을 링크형으로 변경(`:261`): `items: string[]` → `items: { label, to }[]`, `<span>`→`<Link to>`.
  ```tsx
  <FooterCol title="서비스" items={[
    { label: '강의 둘러보기', to: '/library' },
    { label: '전문가', to: '/experts' },
  ]} />
  <FooterCol title="고객지원" items={[
    { label: '문의하기', to: '/contact' },
    { label: '이용약관', to: '/terms' },
    { label: '개인정보처리방침', to: '/privacy' },
  ]} />
  ```
  3칼럼 그리드(`:127`)는 2칼럼으로(`grid-cols-2` 유지/조정). 사업자 정보 블록(`:135-151`)·로고·© 그대로.

## 정책 본문 — 그래플레이에서 가져오기 (구독 제외)

> 디자인은 비즈 라이트 테마(`StaticPage`)로 옮기되 **문구·조항 구조는 그대로 차용**. 다크(slate-950)는 안 씀.

### 이용약관(`/terms`) — `grapplay/pages/Terms.tsx` 기반
- **그대로 가져옴**: 제1조(목적)~제11조(분쟁) + 부칙, **제6조의2(환불정책)**.
- **비즈 조정**:
  - "Grapplay / 온라인 주짓수 교육" → **"그래플레이 비즈 / 체육관 경영자·지도자를 위한 비즈니스 교육"**.
  - 콘텐츠 "강좌·영상·루틴·드릴·스파링" → **"강의·전자책"**, "크리에이터" → **"전문가(지도자)"**.
  - 제5조 서비스 목록 → 강의·전자책·전문가 리뷰로 축소(1:1 피드백/커뮤니티 등 제거).
  - 결제수단 "PayPal·간편결제" → **토스페이먼츠**.
- **❌ 제거(구독)**: 제6조 ③구독 자동갱신·해지, ④연간구독 30% 할인. (단품=평생소장: 결제 1년 유료·이후 무료 유지.)

### 환불정책 — 약관 제6조의2 안에 (별도 `/refund` 없음)
- **그대로 가져옴**: ①개별 콘텐츠 환불(7일 내·미이용 전액, 강의 진도율 10% 미만 전액, 7일/이용 시 불가),
  공통(처리 3~5영업일, 전문가 귀책 시 전액).
- **조정**: "강좌/드릴/스파링" → "강의", **전자책** = "다운로드·열람 기록 없으면 7일 내 전액 환불".
- **❌ 제거**: ②구독 환불, ③번들(미운영), ④보유 포함 번들, ⑤구독자 할인, ⑥1:1 피드백.
