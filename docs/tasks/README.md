### 확정된 결정
- 작업 ①: 카테고리 **"체육관 운영" → "경영"**, DB 컬럼/시드/기존 행까지 변경(A안).
- 작업 ②: 메뉴 "내 강의" → **`/my`**.
- 작업 ⑦: 전자책 **`category` 컬럼 신설**(A안). **모든 페이지 내부 검색 제거 → 통합검색만**.
- **네비**: 상단 메뉴(데스크톱)는 **강의·전자책·전문가 유지**(전문가 드롭다운 이동 취소). 모바일은 **하단 메뉴를
  별도 신설**(홈/검색/컨텐츠/내강의) + 모바일 상단 검색바 제거. **컨텐츠(/content)는 모바일 진입 전용 통합 페이지**.

### 의존/순서 메모
- **카테고리 문자열**(작업 ①) + **전자책 카테고리 컬럼**(작업 ⑦)이 **선행** → ④·⑨의 카테고리 필터 정상화.
- **검색은 통합검색(⑧) 하나로 일원화**: 데스크톱 헤더 검색창 + 모바일 하단 "검색" → `/search`. 페이지 내부 검색 없음.
- 권장 구현 순서: ①·⑦(데이터/카테고리) → ⑥(네비) → ⑨·⑧(허브·검색) → ②③④ 정리.

> 구독제(구독/정기결제)는 비즈에 없으므로, 약관·환불 등 정책 문서에서 **구독 관련 조항은 제외**한다
> (단품 구매 기준). 정책 본문은 **그래플레이 본 프로젝트의 실제 페이지**
> (`../grapplay/pages/Terms.tsx`·`Privacy.tsx`)에서 가져와 구독 부분만 들어낸다 — 매핑은 [03](./03-footer-links-and-policy-pages.md) 참조.
